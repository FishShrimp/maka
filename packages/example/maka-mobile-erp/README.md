使用maka.js + antd-mobile + echarts +  font-awesome 实现简单h5手机端

## 运行

```bash
$ npm i -g @makajs/cli
$ yarn install
$ yarn start 
```
打开浏览器，http://127.0.0.1:8000, 登录用户名：13334445556 密码:1
